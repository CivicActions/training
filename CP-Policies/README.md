# Security Training Program

This project converts CivicActions' 17 security and compliance policies into six role-based training courses delivered as interactive LiaScript modules.

## Directory structure

- **policies/** — Source policy documents exported from Confluence (markdown).
- **outcomes/** — Learning outcome tables for each policy, plus CSV/mapping files linking outcomes to trainings and modules.
- **outlines/** — Training outlines that organize outcomes into audience-specific courses and modules.
- **drafts/** — LiaScript course files (one README.md per training) with quizzes, ready for review and delivery.

| Role | Trainings | Est. total time |
|------|-----------|-----------------|
| General staff | 1 | ~35 min |
| Staff on CUI contracts | 1 + 2 | ~45 min |
| Developers | 1 + 3 + 5 | ~60 min |
| IT / Service Desk | 1 + 3 + 4 | ~60 min |
| SIRT members | 1 + 3 + 4 | ~60 min |
| Managers | 1 + 4 + 6 | ~60 min |
| System / Data Owners | 1 + 3 + 6 | ~70 min |
| Document Controllers | 1 + 6 | ~55 min |

Add Training 2 (CUI) to any role working with federal CUI data.
