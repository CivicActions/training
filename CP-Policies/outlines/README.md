# Training Program — Draft Outlines

This folder contains draft session outlines for the CivicActions security training program. Each training is designed to cover the learning outcomes documented in [`outcomes/`](../outcomes/).

## Design principles

- **Minimal sessions:** 6 trainings cover all 193 outcome rows from 17 policies
- **Smallest audience:** each session targets the narrowest group that needs the content
- **Short delivery:** one primary session (30–40 min) for all staff; role-specific sessions are 10–20 min each
- **No duplication:** an outcome appears in exactly one training; the all-staff session is a prerequisite for all others
- **CUI is opt-in:** only staff on federal/CUI contracts take the CUI module

## Delivery model

- **Format:** short written module + scenario-based quiz; signed acknowledgment doubles as compliance evidence
- **Cadence:** onboarding (within 30 days) + annual refresher + ad-hoc on material policy change
- **Records:** completion + signed acknowledgment stored in HRIS/LMS
- **Compliance:** CMMC L1 (FAR 52.204-21), CMMC L2 (NIST 800-171 AT.L2-3.2.1/3.2.2), ISO 27001:2022 (A.6.3, 7.2, 7.3, A.8.2)
