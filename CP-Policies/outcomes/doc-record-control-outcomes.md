# Document and Record Control Policy — Training Outcomes

Source: [Document and Record Control Policy](../policies/Document-and-Record-Control-Policy_194019329.md)

| Policy | Section | Learning Outcome | Audience |
|--------|---------|-----------------|----------|
| DRCP | Document Identification | Understand how Controlled Documents are identified and tracked in the Controlled Document Jira board; explain the document ID format (two-character department prefix + document type digit + three-digit unique number) | Document Controllers; Managers |
| DRCP | Document Classification | Apply data classification (Public, Internal, Confidential, Restricted) to Controlled Documents using the Data Classification SOP; document the classification in the Document Control Change Log | Document Controllers; System/Data Owners |
| DRCP | Controlled Document Format | Ensure Controlled Documents include required sections: document information (ID, version, effective date, classification), Purpose, Scope, Roles and Responsibilities, References, main section, Training, and Review | Document Controllers; Managers |
| DRCP | Document Version Control | Distinguish major versions (significant content/scope changes) from minor versions (typos, link updates, formatting); understand that effective Controlled Documents are immutable; changes create a new draft | Document Controllers |
| DRCP | Document Change Request Process | Submit document change requests through the Compliance Jira board; understand the workflow from draft distribution through feedback, approval, and distribution of the controlled copy | All Staff |
| DRCP | Access Control and Security | Understand that Controlled Document locations use security groups in Google Workspace, Confluence, and Jira to manage access | All Staff |
| DRCP | Document Training Requirements | Understand that Management and the Responsible Person determine training requirements for each Controlled Document; the Document Controller coordinates training planning | Managers; Document Controllers |
| DRCP | Document Review | Know that all Controlled Documents are reviewed annually by default; documents with different schedules note this in their Review section | All Staff |
